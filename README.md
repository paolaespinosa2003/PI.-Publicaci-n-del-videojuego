# Criterios de evaluación
El videojuego en 3D cuenta con una interfaz de inicio.  
Integra un sistema de navegación funcional para toda la aplicación.  
El entorno en 3D cuenta con una correcta iluminación, materiales y mejoras visuales para una experiencia de usuario.  
Muestra una mecánica principal de juego.  
El videojuego cuenta con al menos tres niveles distintos de dificultad.  
Permite un seguimiento y registro al avance del jugador.  
Integra efectos que enriquecen la experiencia de juego del usuario de la aplicación.    
La dinámica del juego permite mover al personaje en un espacio tridimensional, además de una curva de dificultad progresiva y guardado de datos  
Se puede exportar a un dispositivo móvil de forma funcional.  
Visita y retroalimenta el videojuego de al menos dos compañeros(as). 
La información que se presenta en el videojuego es clara, se cuida la ortografía y la redacción.  
Las herramientas utilizadas para el desarrollo provienen de los sitios oficiales y en su versión más reciente.   
Se crean las cuentas necesarias y se ejecutan los entornos para el desarrollo.
